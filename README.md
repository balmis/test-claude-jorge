# test-claude-jorge

Repositorio de prueba para explorar y experimentar con Claude Code y el Claude Agent SDK.

## Descripción

Este proyecto sirve como entorno de pruebas para:

- Evaluar las capacidades de **Claude Code** como asistente de ingeniería de software
- Experimentar con flujos de trabajo asistidos por IA directamente en el editor
- Probar integraciones con herramientas de desarrollo (GitHub, Figma, etc.)

## Tecnologías

- [Claude Code](https://claude.ai/code) — CLI oficial de Anthropic para Claude
- Git / GitHub

## Uso

Este repositorio es un sandbox personal. No tiene dependencias externas ni requiere instalación especial.
